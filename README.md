# ⏱️ Cronômetro Web

Projeto de um cronômetro desenvolvido com **HTML, CSS e JavaScript**, permitindo iniciar e parar a contagem de segundos através de uma interface moderna e responsiva.

## 🌐 Demonstração

Você pode publicar o projeto no GitHub Pages para disponibilizá-lo online.

## 📋 Funcionalidades

* Iniciar a contagem de segundos.
* Pausar o cronômetro.
* Atualização automática do contador a cada segundo.
* Interface moderna e responsiva.
* Animações nos botões.

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

## 📂 Estrutura do projeto

```bash
📦 cronometro
 ┣ 📜 index.html

 ┗ 📜 README.md
```

## ⚙️ Como funciona

O cronômetro utiliza:

* `setInterval()` para incrementar a contagem a cada segundo.
* `clearInterval()` para interromper a execução.
* Manipulação do DOM com `querySelector()`.
* Atualização dinâmica do conteúdo utilizando `innerHTML`.
